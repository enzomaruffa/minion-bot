# Task Tools

Create Agno tools for task management.

## Acceptance Criteria
- [ ] add_tasks(tasks: list[dict]) - batch create tasks
- [ ] update_task(task_id, updates) - modify task fields
- [ ] list_tasks(status, date_range) - filter tasks
- [ ] search_tasks(query) - full text search
- [ ] get_task_details(task_id) - single task with attachments
- [ ] delete_task(task_id) - soft delete or archive
